백엔드 레포지토리 담당 Claude Code에게 전달할 프롬프트
====================================================================

안녕하세요!

똑간집 배달 서비스의 프론트엔드가 완성되었습니다.
이제 프론트엔드와 연동할 수 있도록 백엔드 API를 구현해주세요.

📁 첨부 파일: BACKEND_API_SPEC.md
이 파일에 모든 API 명세가 상세하게 정리되어 있습니다.

🎯 핵심 요구사항:

1. **API 엔드포인트 구현**
   - 인증 API: 회원가입, 로그인, 토큰 갱신, 로그아웃
   - 메뉴 API: 메뉴 목록, 메뉴 상세 조회
   - 장바구니 API: 조회, 추가, 수정, 삭제
   - 주문 API: 체크아웃, 주문 내역, 주문 상세
   - 직원 API: 활성 주문 조회, 주문 상태 변경

2. **인증 시스템**
   - JWT 기반 인증 (Access Token + Refresh Token)
   - Access Token 만료: 1시간
   - Refresh Token 만료: 7일
   - 역할: USER, STAFF

3. **CORS 설정**
   - 프론트엔드 주소: http://localhost:5173 (개발 서버)
   - credentials: true
   - 헤더: Content-Type, Authorization

4. **서버 포트**
   - 8080 포트로 실행
   - 환경변수로 설정 가능하게

5. **데이터베이스**
   - BACKEND_API_SPEC.md에 스키마 제안 포함
   - 테스트 데이터 시드 필요

📋 작업 순서:

1. BACKEND_API_SPEC.md 파일을 자세히 읽어주세요
2. 데이터베이스 스키마를 생성해주세요
3. API 엔드포인트를 하나씩 구현해주세요
4. 테스트 데이터를 시드해주세요
5. 각 엔드포인트를 테스트해주세요
6. 서버를 8080 포트로 실행해주세요

🧪 테스트 계정:

일반 사용자:
- 이메일: user@test.com
- 비밀번호: test1234

직원:
- 이메일: staff@test.com
- 비밀번호: staff1234

✅ 완료 후:

"백엔드 API 구현이 완료되었습니다. 프론트엔드와 연동 테스트를 진행해주세요."
라고 알려주시면, 프론트엔드 측에서 연동 테스트를 시작하겠습니다.

질문이나 불분명한 부분이 있으면 언제든 물어봐주세요!
